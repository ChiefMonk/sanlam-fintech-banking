# Bank Account Withdrawal Code Improvement Exercise

## Overview

This exercise improves a basic bank account withdrawal flow while preserving the original business capability:

> **Withdraw an amount from an account balance and notify the rest of the system that the withdrawal happened.**

The goal is not to change the business behavior, but to improve the implementation quality across structure, correctness, maintainability, observability, fault tolerance, consistency, and long-term adaptability.

I have included a pdf file, [Sanlam Suggested Improvements](./Sanlam_Suggested_Improvements.pdf).

This document includes:

- an outline of the improvement approach
- rationale behind implementation choices
- architectural and design decision choices
- notes on any unclear or non-obvious library usage
